🛡️ 비트대장(Upbit) 개인정보처리방침

최종 수정일: 2025년 12월 20일

제1조 (수집 항목 및 저장 기술)

(1) 계정 정보: Google 로그인을 통해 제공되는 이메일 주소 및 식별 정보를 수집합니다.
(2) 거래소 API 키: Upbit Access Key 및 Secret Key를 사용합니다.
(3) 기술적 보호: 모든 API 키는 서버로 전송되지 않으며, 사용자의 스마트폰 내부 보안 저장소(FlutterSecureStorage)에 암호화되어 로컬에만 보관됩니다.

제4조 (회원 탈퇴 및 데이터 자동 파기)

사용자는 앱 내 '환경설정' 메뉴를 통해 언제든지 회원 탈퇴를 요청할 수 있습니다. 탈퇴 즉시 Firebase Authentication에 저장된 이메일 기록 및 Firestore의 모든 관련 사용자 데이터는 영구적으로 자동 삭제되며, 기기 내 저장된 API 키 역시 즉각 파기됩니다.

제5조 (제3자 서비스 및 광고 제공)

본 서비스는 맞춤형 보상형 광고 제공을 위해 Google AdMob 서비스를 이용합니다. 이 과정에서 구글은 사용자의 기기 광고 식별자(ADID/IDFA) 등을 수집할 수 있으며, 이는 구글의 개인정보보호정책을 따릅니다.
