🛡️ 비트대장(Upbit) 개인정보처리방침

최종 수정일: 2025년 12월 20일

'비트대장'(이하 '앱')은 이용자의 개인정보를 보호하며, 관련 법규를 준수합니다. 본 방침을 통해 수집되는 정보의 용도와 보호 조치를 안내드립니다.

제1조 (수집 항목 및 방법)

(1) 계정 관리: Google 로그인 시 제공되는 이메일 주소 및 사용자 식별 정보를 수집합니다.
(2) 거래소 연동 정보: 사용자가 직접 입력한 Upbit Access Key 및 Secret Key를 사용합니다.
(3) 보안 강조: 수집된 API 키는 개발자 서버로 전송되지 않으며, 기기 내부 보안 저장소(FlutterSecureStorage)에 암호화되어 로컬에만 보관됩니다.

제2조 (이용 목적)

수집된 정보는 사용자 식별, 자동매매 명령 실행, 자산 및 거래 내역 조회 등 서비스의 핵심 기능을 제공하기 위해서만 활용됩니다.

제7조 (안전성 확보 조치)

앱은 민감한 API 정보를 기기 내 하드웨어 보안 영역에 격리 저장하며, 모든 통신은 SSL 암호화(HTTPS)를 통해 안전하게 처리됩니다.
