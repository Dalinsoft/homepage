🛡️ 비트대장(Upbit) 서비스 이용약관

제1조 (목적)

본 약관은 사용자가 비트대장(이하 "서비스")을 기술적 매매 지표를 학습하고 연구하는 보조 도구로 사용하는 것과 관련하여 필요한 사항을 규정함을 목적으로 합니다.

제2조 (용어의 정의)

* "서비스": 사용자의 Upbit 거래소 계정을 연동하여 사용자가 직접 설정한 지표 조합에 따라 현물 거래 명령을 자동 수행하는 유틸리티 소프트웨어입니다.
* "API 키": Upbit 계정 접근을 위해 필요한 Access Key와 Secret Key를 의미하며, 사용자가 직접 발급받아 서비스에 입력해야 합니다.

제5조 (서비스의 범위 및 제한)

* 본 서비스는 업비트(Upbit) 현물 시장(KRW 마켓 등)을 지원하며, 레버리지 기능을 제공하지 않습니다.
* 사용자의 기기 성능 및 네트워크 상태에 따라 데이터 수신 및 주문 속도에 차이가 발생할 수 있습니다.

제7조 (사용자의 의무)

* 사용자는 API 키 발급 시 반드시 **[자산조회, 주문조회, 원화거래]** 권한만 허용해야 하며, **[출금]** 권한은 반드시 비활성화해야 합니다.
* API 키 관리 소홀로 발생하는 모든 책임은 사용자 본인에게 있습니다.

제8조 (면책조항 및 위험 고지)

⚠️ 본 서비스는 금융 자문 서비스가 아니며, 수익을 보장하지 않는 기술적 연구 보조 도구입니다.
* 모든 매매 전략과 파라미터는 사용자가 직접 설정합니다. 서비스는 사용자의 수동적 설정값을 기반으로 명령을 실행할 뿐이며, 그 결과에 대해 책임을 지지 않습니다.
* 암호화폐 투자는 원금 손실 위험이 매우 큽니다. 모든 투자 결정과 그에 따른 결과는 전적으로 사용자 본인의 책임입니다.
* 거래소 서버 점검, 네트워크 장애, 천재지변 등으로 인한 손해에 대해 서비스는 면책됩니다.
