* "서비스": 사용자의 Bitget 거래소 계정을 연동하여 사용자가 선택한 전략에 따라 선물 매매 명령을 자동 실행하는 소프트웨어입니다.
* "API 키": Bitget 거래소 접근을 위해 필요한 API Key와 Secret Key, Passphrase를 의미합니다.
